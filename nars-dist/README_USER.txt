You can launch NARS from command window (shell) in several ways:

- empty reasoner:
java -jar NARS.jar

- reasoner loaded from an experience file:
java -jar NARS.jar nars-dist/Examples/Example-NAL1-in.txt --silence 90

- reasoner in batch:
java -cp NARS.jar nars.main_nogui.NARSBatch nars-dist/Examples/Example-NAL1-in.txt


Example experience files are in directory Examples.
