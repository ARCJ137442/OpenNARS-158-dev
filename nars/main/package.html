Top-level classes of the system

Package Specification

This package contains the top classes of the system.

NARS defines the application and applet. When NARS runs as an applet, it is for demonstration only, and certain functionalities are blocked.

Parameters collects all system parameters, which can be edited before compiling.

Memory maintains the work space of the system, including

  • a bag of concept,
  • a look-up table of all built-in operators,
  • a queue for new results to be processed in each cycle,
  • a task buffer for new results to be processed in the future.
Each of the above occupies constant space.

Memory and all of its variables are static, so each is unique, and can be referred by name anywhere in the system.
